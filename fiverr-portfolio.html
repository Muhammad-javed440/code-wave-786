Project Overview

Code Wave AI is a premium technology brand platform built with React 19, TypeScript, and Supabase. It features role-based authentication, an admin dashboard with analytics, a public project showcase with user interactions, AI voice chatbot integration, and a fully responsive dark/light mode UI. Deployed with Docker and Nginx for production-ready performance.
